# DCARO PROGRESS - Automatización de Procesos

Landing page interactiva para **DCARO PROGRESS** orientada a la estructuración y automatización de procesos de negocio utilizando las herramientas existentes de Microsoft 365.

## 🚀 Características
- **Diseño Premium**: Interfaz moderna, limpia y altamente responsiva con micro-animaciones y efectos visuales de alta calidad.
- **Simulador Interactivo**: Comparativa visual entre la operación tradicional con caos y la operación automatizada con DCARO PROGRESS.
- **Diagnóstico Multi-paso**: Formulario interactivo de 3 pasos que ayuda a perfilar los problemas de la empresa.
- **Integración con WhatsApp**: Envía el diagnóstico detallado directamente a la línea de contacto corporativo.
- **Cifras de Impacto Animadas**: Contadores en tiempo real que ilustran métricas de eficiencia.

## 🛠️ Tecnologías
- HTML5 (Estructura semántica)
- CSS3 (Diseño responsivo, Variables CSS, Glassmorphism, Animaciones)
- JavaScript (Lógica interactiva, Animación de contadores, Formulario multi-paso)
